# 🎓 MediQueue — Advanced Tutor Booking & Scheduling Platform

MediQueue is a modern, full-stack ed-tech web application designed to seamlessly connect students with expert tutors. It eliminates manual scheduling conflicts through an automated slot booking engine and digital session token system.

## 🚀 Key Features

- 🔍 **Multi-Parametric Filtering & Search:** Search tutors by name, subject, teaching mode (Online/Offline/Both), and real-time Price Sorting (Low to High / High to Low).
- 🖼️ **Interactive Image Lightbox:** Full-screen zoom modal for tutor profile pictures without awkward cropping.
- 🎟️ **Conflict-Free Session Token Engine:** Generates unique digital booking tokens upon successful session confirmation.
- 📱 **100% Pixel-Perfect & Responsive:** Built with Tailwind CSS v4, aligning seamlessly with modern Figma UI/UX specs.
- ⚙️ **Full CRUD Operations:** Tutors can create, update, and manage their profile listings; students can track and cancel booked appointments.
- 🔙 **Smart Browser History Navigation:** Supports native browser Back/Forward navigation (`<-` / `->`) via window location hashes without full-page reloads.
- ⚡ **Skeleton Loading States:** Animated shimmer effects during API data fetching for enhanced UX.

## 🛠️ Tech Stack

### **Frontend:**
- **Core:** React.js (Vite)
- **Styling:** Tailwind CSS v4
- **Icons & Animations:** Lucide React, Framer Motion
- **Notifications:** SweetAlert2
- **API Client:** Axios

### **Backend:**
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB & MongoDB Atlas
